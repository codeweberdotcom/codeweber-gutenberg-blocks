# CodeWeber Gutenberg Blocks · Документация

Набор рабочих инструкций для разработки и рефакторинга плагина `codeweber-gutenberg-blocks`. Структура вдохновлена `ROOT_FILES_DESCRIPTION.md` из Materio, но адаптирована под WordPress и Gutenberg.

| Файл | Назначение |
|------|------------|
| `QUICK_START.md` | Быстрый старт: окружение, запуск, обязательные проверки |
| `DEV_WORKFLOW.md` | Полный цикл задачи: анализ → план → реализация → тест → релиз |
| `GUTENBERG_BLOCK_STANDARDS.md` | Дополнительные требования к блокам Gutenberg |
| `REFRACTORING_GUIDE.md` | Чек-лист для безопасного рефакторинга и миграций |

**Как использовать**
1. Начинаем с `QUICK_START.md`, чтобы привести окружение и WP-сайт к рабочему состоянию.
2. Для любой задачи следуем фазам из `DEV_WORKFLOW.md`.
3. Работая с блоками Gutenberg, обязательно сверяемся с `GUTENBERG_BLOCK_STANDARDS.md`.
4. Перед слиянием крупных правок проходим чек-лист из `REFRACTORING_GUIDE.md`.

> Любые новые инструкции или специализированные модули добавляем как отдельные файлы внутри `doc/` и регистрируем их в таблице выше.
